## JavaWeb--HTML

- 特殊字符

  - 常用的特殊字符

    ```
    <  ====>>>  &lt; less than
    > 	===>>>  &gt; great than
    space &nbsp; 空格
    
    ```

- 超链接

  - ```
    a标签是超链接
    	href 属性设置链接的地址
    	target 属性设置哪个目标进行跳转
    		_self 表示当前页面
    		_blank 表示当前打开新页面来进行跳转
    <a herf="http://localhost:8080" target="_self">百度</a>
    ```

- 表格标签

  - border 设置表格标签
  - width   设置表格宽度
  - height   设置表格高度
  - tr       是行标签
  - th      是表头标签
  - td      是单元格标签
  - align 设置单元格文本对齐方式
  - b    是加粗标签
